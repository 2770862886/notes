第一篇文档
==========

``` latex
\documentclass{article}
% 这里是导言区
\begin{document}
Hello, world!
\end{document}
```

``` latex
\documentclass{article}
\usepackage{xeCJK} %调用 xeCJK 宏包
\setCJKmainfont{SimSun} %设置 CJK 主字体为 SimSun （宋体）
\begin{document}
你好，world!
\end{document}
```
